# Master Inventory Template

The Master Inventory is the Single Source of Truth (SSOT).

Only the user may modify the fields in the Identity table.

The AI maintains the Market Data fields.

Mark any uncertain value as **Needs Confirmation** — never guess silently.

## Field Reference

- **Category**: Basketball / Baseball / Soccer / Pokémon / MTG / Marvel / Entertainment / Other
- **Confidence**: High / Medium / Low (Confidence Reason mandatory when not High)
- **Opportunity Score**: 1–100 (90–100 Strong Buy, 80–89 Buy, 65–79 Hold, 50–64 Watch, <50 Consider Selling)
- **Tier**: S Core Holdings / A High Growth / B Hold-Watch / C Sell-Rotate
- **Action**: Hold / Buy / Sell / Watch
- **Investment Thesis**: required for S and A tiers
